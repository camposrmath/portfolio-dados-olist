# 📊 Portfólio de Análise de Dados — Olist E-commerce

Olá, recrutadores e afins!

Este portfólio contém projetos de análise de dados construídos sobre o dataset público da **Olist** (Kaggle), com ~100 mil pedidos reais do e-commerce brasileiro entre 2016 e 2018.

Espero que goste dos projetos e qualquer coisa, só me chamar! :)

---

## 🗂️ Projetos

| # | Projeto | Ferramenta | Técnicas |
|---|---------|------------|----------|
| 1 | KPIs Básicos de Vendas | SQL | JOINs, GROUP BY, agregações |
| 2 | Análise Avançada | SQL | CTEs, Window Functions, subqueries |
| 3 | Dashboard Executivo | Excel | Tabelas, gráficos, fórmulas entre abas |
| 4 | Segmentação RFM | Python | Pandas, Matplotlib, Seaborn |
| 5 | Dashboard 360° | Power BI | Star Schema, DAX, mapas |
